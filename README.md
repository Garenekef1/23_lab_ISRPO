Проект содержит игру крестики-нолики и страницу профиля пользователя.
Папка img используется для скриншотов по шагам лабораторной.

Описание игры:
Игра крестики-нолики на поле 3x3 для двух игроков.
Игроки ходят по очереди символами X и O.
Победа определяется при совпадении символов по горизонтали, вертикали или диагонали.
Если все клетки заполнены и победителя нет, фиксируется ничья.

Игровая механика:
Победитель определяется проверкой массива board по 8 выигрышным комбинациям.
Клики по клеткам обрабатываются функцией handleCellClick.
Если клетка занята или игра закончена, ход не выполняется.
Кнопка перезапуска сбрасывает массив поля, текущего игрока и интерфейс.

Структура данных:
board - массив из 9 элементов, хранит X, O или пустую строку.
winningConditions - массив выигрышных комбинаций индексов.
gameActive - флаг активности игры.
currentPlayer - текущий игрок.

Алгоритмы:
Проверка победы: перебор winningConditions и сравнение трёх значений.
Проверка ничьи: отсутствие пустых строк в board.
Смена игрока: тернарный оператор после каждого валидного хода.
